Esta carpeta contiene los resultados exportados de las consultas SQL en formato CSV.

## Contenido

Los archivos incluyen información como:

- Ventas totales
- Ventas netas
- Utilidad
- Devoluciones
- Órdenes
- Ventas por país
- Ventas por subcategoría
- Ventas por mes
- Productos con mayores ingresos
- Margen de ganancia
