Esta carpeta contiene los dashboards desarrollados en Power BI.

## Contenido

- Archivo Power BI (.pbix)
- Versión en PDF del dashboard (opcional)

El dashboard presenta indicadores clave (KPIs) y visualizaciones diseñadas para facilitar el análisis del desempeño comercial.

Entre los análisis incluidos se encuentran:

- Ventas totales
- Ventas netas
- Utilidad
- Devoluciones
- Ventas por país
- Ventas por subcategoría
- Tendencia mensual
- Productos con mayores ingresos
