# 📊 Sales Analysis Dashboard | AdventureWorks

## 📌 Descripción

Este proyecto consiste en un análisis de ventas desarrollado con **MySQL** y **Power BI** utilizando la base de datos **AdventureWorks**.

El objetivo fue analizar el desempeño comercial de la empresa durante los años **2020, 2021 y 2022**, transformando datos transaccionales en información útil para evaluar ventas, rentabilidad, devoluciones, categorías de productos y mercados con mayor generación de ingresos.

---

## 🎯 Objetivos del análisis

El proyecto responde las siguientes preguntas de negocio:

- ¿Cuáles fueron los ingresos totales de la empresa?
- ¿Cuál fue la utilidad generada durante el periodo analizado?
- ¿Cuántas órdenes fueron realizadas y cuántas fueron canceladas?
- ¿Cuál fue el impacto económico de las devoluciones?
- ¿Cómo evolucionaron las ventas entre 2020 y 2022?
- ¿Qué países generan la mayor cantidad de ingresos?
- ¿Qué categorías y subcategorías concentran la mayor parte de las ventas?
- ¿Qué productos presentan los mejores márgenes de utilidad?
- ¿Cómo afectan las devoluciones a las ventas netas y a la utilidad?

---

## 📈 Hallazgos principales

- De un total de **56,046 órdenes**, únicamente **1,809** fueron devueltas, representando aproximadamente **3.23%** del total de pedidos.
- De **$24,914,586.82** en ventas, solo **$765,277.84** fueron devueltos (**3.07%** de los ingresos), reflejando una baja tasa de devoluciones.
- Todos los productos presentan un **margen de utilidad superior al 20%**, indicando una rentabilidad saludable en todo el catálogo.
- De las **cuatro categorías** disponibles, únicamente **tres registran ventas**. La categoría restante no presenta movimiento y requiere seguimiento para determinar si corresponde a una nueva línea de productos o a una categoría con baja demanda.
- El análisis comprende los años **2020, 2021 y 2022**. Aunque los datos de **2022 son parciales**, las ventas acumuladas prácticamente igualan las obtenidas durante todo **2021**, reflejando un crecimiento sostenido del negocio.
- **Estados Unidos** es el mercado con mayor generación de ingresos, acumulando **$7,930,759.10** en ventas durante el periodo analizado.
- Las subcategorías con mayor generación de ingresos son:
  - Road Bikes
  - Mountain Bikes
  - Touring Bikes
  - Tires and Tubes
  - Helmets

## 📊 Dashboard

El dashboard fue desarrollado en **Power BI** y está dividido en diferentes secciones para facilitar el análisis de los principales indicadores comerciales.

Incluye visualizaciones de:

- KPIs generales
- Ventas netas por año
- Ventas por país
- Ventas por categoría
- Utilidad por año
- Costo neto por año
- Tendencia de ventas
- Devoluciones
- Órdenes y cancelaciones
